# Useful Tools

一个基于 React + TypeScript + Vite 的在线工具集合，提供多种实用的开发工具。

## 功能列表

- **Base64 转换器** - 文本与 Base64 编码相互转换
- **Diff 检查器** - 比较两段文本的差异
- **图片压缩器** - 压缩图片文件大小
- **图片转 PDF** - 将图片转换为 PDF 文件
- **JSON 格式化** - 美化和格式化 JSON 数据
- **JWT 解码器** - 解码和查看 JWT Token 内容
- **密码生成器** - 生成高强度随机密码
- **PDF 转图片** - 将 PDF 文件转换为图片
- **SVG 转换器** - SVG 与其他格式相互转换
- **URL 编码器** - 编码和解码 URL 字符串

## 技术栈

- React 19
- TypeScript
- Vite 7
- Lucide React (图标)
- jsPDF (PDF 生成)
- JSZip (压缩包处理)
- pdf.js (PDF 解析)
- diff-match-patch (差异比较)

## 快速开始

### 安装依赖

```bash
pnpm install
```

### 开发模式

```bash
pnpm run dev
```

### 构建生产版本

```bash
pnpm run build
```

### 预览构建结果

```bash
pnpm run preview
```

### 代码检查

```bash
pnpm run lint
```
